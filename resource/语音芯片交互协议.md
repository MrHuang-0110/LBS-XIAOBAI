# 小白-Ai MCU ↔ ASRPRO UART 交互协议 v0.3

- 更新日期：2026-07-03（v0.3：分隔符改为不对称、数值统一十进制）
- 物理层：UART 115200 8N1 全双工
- 目标 MCU：PY32F030K28U6TR
- 语音芯片：ASRPRO
- 最大帧长：**32 字节**（含 `\r\n`）

---

## 一、帧格式

ASCII 文本，`\r\n` (0x0D 0x0A) 作为帧结束符。

**MCU → ASRPRO：** 使用 `=` 作为 tag 与数值的分隔符
```
<tag>[=<dec>]\r\n
```

**ASRPRO → MCU：** 使用 `:` 作为 tag 与数值的分隔符
```
<tag>[:<dec>]\r\n
```

- `tag`：全小写英文字母，长度 ≤ 8（例如 `play`、`cmd`、`wake`、`done`、`stop`、`ping`）
- `dec`：**十进制**数值字符串（可以 `1` 或 `01` 或 `10` 都接受）
- 结束符：`\r\n` 两字节

**约束：**
- 收方在遇到 `\r\n` 时才判定为一个完整帧
- 单帧总长超过 32 字节 → 收方直接丢弃当前缓冲，重新开始收帧
- 未识别的 tag、格式非法、数值超出 `0..255` → 收方忽略该帧（不 NACK 不响应）

## 二、命令表

### MCU → ASRPRO

| 帧文本              | 语义         | 说明                        |
| ------------------ | ------------ | --------------------------- |
| `play=<id>\r\n`    | 播报预置语音 | id = 语音 ID（见第三节）   |
| `stop\r\n`         | 停止当前播报 | —                           |
| `ping\r\n`         | 心跳/存活探测 | —                           |

### ASRPRO → MCU

| 帧文本              | 语义         | 说明                        |
| ------------------ | ------------ | --------------------------- |
| `cmd:<id>\r\n`     | 语音命令识别 | id = 命令 ID（见第四节）   |
| `wake\r\n`         | 检测到唤醒词 | —                           |
| `done:<id>\r\n`    | 播报完成     | id = 上次的语音 ID          |

## 三、语音 ID 表（十进制）

```
01 开机语     02 关机语     03 进入动力
04 进入感应   05 进入遥控   06 进入语音
07 遥控连接   08 遥控断开   09 低电量
10 收到       11 前进 12 后退 13 左转 14 右转 15 停止
20 靠近启动   21 遇障停止   22 挥手开关   23 明暗调速
```

## 四、命令 ID 表（十进制）

```
30 前进 31 后退 32 左转 33 右转 34 停止
40 左电机正转 41 左电机反转 42 左电机停止
43 右电机正转 44 右电机反转 45 右电机停止
50 进入动力 51 进入感应 52 进入遥控 53 进入语音
```

（说明：本表已按语音端实际实现精简 —— 风扇/投石机模型玩法暂缓，未来加时会往 60+ 段续。）

## 五、示例

**MCU 请求播开机语：**
```
play=01\r\n
```
（hex: `70 6C 61 79 3D 30 31 0D 0A`）

**ASRPRO 识别到"前进"：**
```
cmd:30\r\n
```

**ASRPRO 上报开机语已播完：**
```
done:01\r\n
```

## 六、变更记录

- **v0.3 (2026-07-03)**：
  - MCU→ASR 分隔符从 `:` 改为 `=`（配合 ASRPRO 端图形化实现）
  - 数值编码从 hex 统一为 **dec**（避免 hex/dec 混用造成 bug）
  - 命令 ID 表精简：删掉未实现的 0x60/0x70 段（风扇/投石机）
- **v0.2 (2026-07-03)**：从定长 6 字节二进制改为 ASCII 文本，冒号分隔、hex 数值。已废弃。
- **v0.1 (2026-07-03)**：初版二进制帧 `A5 03 CMD D0 D1 XOR`。已废弃。
