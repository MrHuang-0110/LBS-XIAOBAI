# 小白机器人 功能定义与设计文档 V1.8

更新日期：2026-06-01  
阶段：量产前规划  
形态：机器人头部形象的智能积木动力盒

## 1. 产品概述

  小白机器人用于儿童科教积木，可控制两个电机，支持语音、感应、遥控和积木扩展。

  核心功能：

  - 双电机控制
  - 主机按键控制
  - 普通话语音控制
  - 红外感应玩法
  - 遥控器/手机遥控
  - 眼睛 LED 与模式 LED 状态提示

## 2. 套装规划

  | 版本 | 包装形态 | 配套内容 | 定位 |
  | --- | --- | --- | --- |
  | 基础版 | 盒装 | 积木、主机、1 个马达、说明书 | 入门款，控制成本，适合基础动力和语音玩法 |
  | 进阶版 | 积木桶装 | 积木、拓展积木、主机、2 个马达、遥控器、遥控器充电线、说明书 | 完整体验款，支持双电机、遥控和更多搭建玩法 |

  | 区域 | 内容 |
  | --- | --- |
  | 正面 | 眼睛 LED、红外对管 |
  | 顶部 | 动力键、感应键、遥控键、语音键、4 个模式 LED |

## 4. 模式

  | 模式 | 进入方式 | 主要功能 |
  | --- | --- | --- |
  | 动力模式 | 动力键 / 语音 | 主机按键控制前进、后退、转向、停止 |
  | 感应模式 | 感应键 / 语音 | 主机按键切换基础感应玩法 |
  | 遥控模式 | 遥控键 / 语音 | 遥控器或手机控制 |
  | 语音模式 | 开机默认 / 语音键 / 语音 | 执行所有语音控制 |

  开机默认进入：**语音模式**

## 5. 按键

  | 按键 | 操作 | 功能 |
  | --- | --- | --- |
  | KEY1（语音键） | 长按 | 开机/关机 |
  | KEY1（语音键） | 短按 | 进入语音模式 |
  | KEY2（感应键） | 短按 | 进入感应模式 |
  | KEY3（遥控键） | 短按 | 进入遥控模式 |
  | KEY4（动力键） | 短按 | 进入动力模式；已在动力模式时切换动作（停止→前进→后退→左转→右转→停止） |

  > 2026-07-06 变更：原设计 KEY1 兼具循环切模式，现改为 4 键各管一个模式，更直观。IO 映射：KEY1=PB3 / KEY2=PB4 / KEY3=PB5 / KEY4=PB8。

## 6. 模式 LED

  | LED | 对应模式 | 对应按键 |
  | --- | --- | --- |
  | LED 1（PB2） | 语音模式 | KEY1 |
  | LED 2（PA10） | 感应模式 | KEY2 |
  | LED 3（PA11） | 遥控模式 | KEY3 |
  | LED 4（PA12） | 动力模式 | KEY4 |

  > 2026-07-06 变更：LED 与按键一一对应。原设计 LED1=动力，现改为 LED1=语音 / LED4=动力。

## 7. 语音播报

  | 场景 | 播报 |
  | --- | --- |
  | 开机 | 你好呀，我是小白，进入语音模式 |
  | 关机 | 再见啦，小白去休息了 |
  | 进入动力模式 | 进入动力模式 |
  | 进入感应模式 | 进入感应模式 |
  | 进入遥控模式 | 进入遥控模式，等待连接 |
  | 进入语音模式 | 进入语音模式 |
  | 遥控连接成功 | 连接成功，可以出发啦 |
  | 遥控连接断开 | 连接断开 |
  | 低电量 | 电量低，请更换电池 |
  | 语音动作确认 | 收到 |
  | 动力模式动作切换 | 播报当前动作：前进/后退/左转/右转/停止 |
  | 感应模式玩法切换 | 播报当前玩法：靠近启动/遇障停止/挥手开关/明暗调速 |

## 8. 动力模式

  动力模式下，短按动力键循环切换动作，并播报当前动作：

  ```text
  停止 → 前进 → 后退 → 左转 → 右转 → 停止
  ```

  | 动作 | L/M1 | R/M2 |
  | --- | --- | --- |
  | 停止 | 停止 | 停止 |
  | 前进 | 正转 | 正转 |
  | 后退 | 反转 | 反转 |
  | 左转 | 停止 | 正转 |
  | 右转 | 正转 | 停止 |

## 9. 感应模式

  感应模式下，短按感应键循环切换玩法，并播报当前玩法：

  ```text
  靠近启动 → 遇障停止 → 挥手开关 → 明暗调速 → 靠近启动
  ```

  | 玩法 | 规则 |
  | --- | --- |
  | 靠近启动 | 检测到物体启动电机，离开后停止 |
  | 遇障停止 | 小车前进，检测到障碍后停止 |
  | 挥手开关 | 每次有效靠近切换开/关 |
  | 明暗调速 | 根据反射强弱切换电机速度 |

## 10. 遥控模式

  | 状态 | 提示 |
  | --- | --- |
  | 等待连接 | 眼睛 LED 慢闪 |
  | 连接成功 | 眼睛 LED 常亮 |
  | 连接断开 | 眼睛 LED 慢闪 |

  遥控模式支持：

  - 组合移动
  - 单电机控制
  - 3 档速度

## 11. 遥控器

  | 区域 | 功能 |
  | --- | --- |
  | 十字键 | 前进、后退、左转、右转，松开停止 |
  | 绿色键 | L/M1 正转 |
  | 红色键 | L/M1 反转 |
  | 蓝色键 | R/M2 正转 |
  | 黄色键 | R/M2 反转 |
  | 肩键 | 速度 + / 速度 - |

  速度档位：1 档 40%、2 档 70%、3 档 100%。默认 2 档。

## 12. 语音模式

  只有语音模式响应具体语音控制。其他模式只响应模式切换语音。

  ### 组合动作

  | 语音 | 动作 |
  | --- | --- |
  | 小白前进 | 双电机前进 |
  | 小白后退 | 双电机后退 |
  | 小白左转 | 左转 |
  | 小白右转 | 右转 |
  | 小白停止 | 停止 |

  ### 单电机

  | 语音 | 动作 |
  | --- | --- |
  | 左电机正转 | L/M1 正转 |
  | 左电机反转 | L/M1 反转 |
  | 左电机停止 | L/M1 停止 |
  | 右电机正转 | R/M2 正转 |
  | 右电机反转 | R/M2 反转 |
  | 右电机停止 | R/M2 停止 |

  ### 模型玩法

  模型玩法是针对特定搭建模型设计的专属语音控制，例如风扇、投石机等。

  | 语音 | 动作 |
  | --- | --- |
  | 进入风扇玩法 | 切换到风扇玩法 |
  | 打开风扇 | 启动风扇电机 |
  | 关闭风扇 | 停止风扇电机 |
  | 进入投石机玩法 | 切换到投石机玩法 |
  | 发射 | 执行投石机发射 |

## 13. 接口

  | 接口 | 功能 |
  | --- | --- |
  | L/M1 | 左侧/一号电机 |
  | R/M2 | 右侧/二号电机 |
  | Type-C | 扩展接口 |

  电机接口：`PH2.0 4pin`

  Pin 定义：

  - pin1：空
  - pin2：电机线 A
  - pin3：电机线 B
  - pin4：空

## 14. 红外对管

  用途：

  - 检测前方物体
  - 检测反射光变化

  红外窗口需要考虑遮光和环境光干扰。

## 15. 供电与 Type-C

  主机供电：`3 x AA`

  Type-C 当前为扩展口。

  当前 AA 电池版本需说明：

  **本版本不支持通过 Type-C 给 5 号电池充电。**
